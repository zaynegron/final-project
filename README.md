# Portfolio Final Project (CSC-2001)

## Goal
Publish a personal portfolio using HTML, CSS, JavaScript, and at least one API/AJAX feature, deployed with GitHub Pages.

## Checklist
- [ ] Replace text with your name and info
- [ ] Add at least **2 projects** (cards on Projects page)
- [ ] One **API/AJAX** feature (e.g., quote, weather, GitHub repos)
- [ ] Responsive layout (test at 360px, 768px, 1280px)
- [ ] Lighthouse score ≥ 90 for Performance & Best Practices
- [ ] Deployed with GitHub Pages
- [ ] Put your live URL in Repo **About → Website** and README

## Deploy (GitHub Pages)
1. **Settings → Pages** → Source: `Deploy from a branch`, Branch: `main`, Folder: `/ (root)`
2. Wait ~1 min. Your site will be live at:
   - If this repo is named `username.github.io` (your personal account): `https://username.github.io`
   - Otherwise (in this org): `https://CSC2001-Fall25.github.io/<repo-name>`

## API Ideas
- GitHub API (list your repos)
- Quotable (quotes), Open-Meteo (weather), TMDB (movies), PokéAPI (fun)
